# Kontextdiagramm

Das Kontextdiagramm beschreibt das System auf höchster Ebene und zeigt:

- welche externen Akteur:innen existieren  
- welche externen Systeme angebunden sind  
- welche Interaktionen stattfinden  

## Diagramm (textuelle Vorform)

```
+-------------+         +-----------------------+         +------------------+
| Nutzer:in   | <--->   |   Unser System        | <--->   | Externes System  |
+-------------+         +-----------------------+         +------------------+
```

> Ein tatsächliches Diagramm kann bei Bedarf später ergänzt werden (Mermaid, PlantUML, Draw.io).
